# spin-wheel-of-names

A high-performance, interactive, and visually stunning **Spin the Wheel** utility. Built with **Next.js 15**, **Tailwind CSS**, and **Framer Motion**, this project features a modern glassmorphism UI and realistic physics.

[🚀 View Live Demo](https://spindewheel.com)

---

## ✨ Key Features

* **💎 Glassmorphism UI:** Sleek, frosted-glass effects with vibrant 3D satin textures.
* **🎯 Precise Physics:** Realistic momentum-based spinning and friction.
* **📱 Fully Responsive:** Optimized for mobile, tablet, and desktop (perfect for TikTok/Shorts content).
* **🔊 Immersive Audio:** Integrated "clicking" sound effects and winner fanfares.
* **⚙️ Highly Customizable:** Easily modify segments, colors, and gravity via a simple config file.
* **⚡ SEO Optimized:** Ultra-fast loading speeds for better search engine ranking.

---

## 🛠️ Tech Stack

* **Framework:** [Next.js 15](https://nextjs.org/) (App Router)
* **Styling:** [Tailwind CSS](https://tailwindcss.com/)
* **Animations:** [Framer Motion](https://www.framer.com/motion/)
* **Deployment:** Optimized for Vercel / Cloudflare Pages
